# Proiect de licenta

## Structura
<br>
Strucutra proiectului implica o arhitectura a aplicatiei ce are la baza ca si stack de tehnologii limbajul Java si cateva dintre framework-urile acestuia, in speta Vaadin, Spring, Hibernat, JPA si o baza de date MySQL.

## Descriere 
<br>
Aplicatia are ca scop imbunatatirea cunostintelor utilizatorilor, in functie de domeniile pe care acestia si le aleg, prin diferite sesiuni de training precum si sesiuni de testare a cunostintelor, cu posibilitatea de a avansa ca si nivel de dificultate.
